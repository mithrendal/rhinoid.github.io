1993 - Mindwarp demo (Amiga)

Probably still high of creating the previous demo, 3 months later we released yet another AGA demo. This was called Mindwarp and had some other effects we wanted to show off. I can remember trying to get the bounce of the balls on the music just right which took quite some time. Especially since the ground could curve, it looked very odd in the first few iterations.

Like the previous demo, I coded all the effects (and like always all in assembly), music was done by Ramon Braumuller and all graphics were made by Metin seven

My appologies for the youtube movie. It’s very blurry. I’ll see if I can grab/upload a high def one at some point.

The demo disk image can be downloaded on the pouet demoscene site
